------------------------------------------------------------------------
       CONTROLE DE CONCORRÊNCIA EM ACESSO A BANCO DE DADOS
------------------------------------------------------------------------

PURPOSE OF PROJECT: O propósito do projeto é trabalhar as classes 
                    de concorrência e paralelismo em java. Foram usados
                    também, semáforos barreiras e monitores para garantir
                    a exclusão mútua das operações de leitura e escrita no
                    banco de dados.

VERSION or DATE: 13/07/2025

AUTHORS: Vinícius Vasconi Villas Boas Micska

USER INSTRUCTIONS: - Insira o número de leitores permitidos simultâneamente.
                   - O Programa instanciará uma série de leitores e escritores
                     executará todos simultaneamente exibindo mensagens sobre o
                     que está acontecendo. 
                   - Ao final o programa exibirá os retornos assíncronos dos
                     leitores e o estado final da tabela.
                   - Valores de constantes podem ser alterados no Main.java

NOTE: Acompanha um executável e um arquivo de diagrama de classes para consulta.
